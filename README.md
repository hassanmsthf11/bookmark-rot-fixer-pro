# Bookmark Rot Fixer Pro

Professional Chrome extension to scan, repair, and maintain your bookmarks automatically.

## Features

- Fix broken bookmark redirects
- Detect and remove duplicate bookmarks
- Find stale and unused bookmarks
- Clean empty folders
- Backup and restore bookmarks
- Export results (CSV / JSON)
- Schedule automatic scans
- Privacy-focused (runs locally)

## Interface

Modern dark UI with:

- Scan dashboard
- Duplicate detector
- Stale bookmark finder
- Backup manager
- Quick Fix automation

## Tech Stack

- JavaScript
- HTML
- CSS
- Chrome Extension Manifest V3
